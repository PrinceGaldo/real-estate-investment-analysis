# Luxury Residential Real Estate Feasibility Analysis

## Overview

This independent research project evaluates the financial feasibility of acquiring luxury residential properties for rental operations.

The study compares two high-end residential properties in Laguna using institutional real estate underwriting techniques.

## Topics Covered

- Net Operating Income (NOI)
- Cap Rate
- Internal Rate of Return (IRR)
- Cash-on-Cash Return
- Equity Multiple
- 10-Year Pro Forma
- Exit Analysis
- Sensitivity Analysis
